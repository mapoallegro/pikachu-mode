# pikachu-mode

A communication mode plugin that replaces filler, pleasantries, and hedging with Pikachu vocabulary — while keeping full technical accuracy.

## Skills

- **pikachu** — Pokémon-flavored communication mode. Supports intensity levels: lite, full (default), ultra, pokemon. Activate with "pikachu mode", `/pikachu`, or "talk like pikachu". Deactivate with "stop pikachu" or "normal mode".
